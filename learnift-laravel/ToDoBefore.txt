Step 1 — Make sure XAMPP is running
Open XAMPP Control Panel and click Start next to both Apache and MySQL.


Step 2 — Check your PHP version
Open your browser and go to:
http://localhost/dashboard/phpinfo.php
Look for the PHP version at the very top of the page. It must be 8.2 or higher.


Step 3 — Install Composer
Composer is required to install Laravel's dependencies.

Go to getcomposer.org/download
Download and run Composer-Setup.exe
During installation, when it asks for the PHP executable path, enter:

   C:\xampp\php\php.exe

Complete the installer — leave all other options as default
Verify it worked by opening Command Prompt and typing:

   composer --version
You should see something like Composer version 2.x.x


Step 4 — Enable required PHP extensions

In XAMPP Control Panel, click Config next to Apache → choose PHP (php.ini)
The file opens in Notepad. Press Ctrl+F and search for each line below. If the line starts with a semicolon ;, remove it to uncomment it:

ini
extension=pdo_mysql
extension=mbstring
extension=openssl
extension=fileinfo
extension=curl
extension=zip
extension=tokenizer	|
extension=xml		| if you cannot find any of these do not worry about it and continue
extension=ctype		|

Save the file (Ctrl+S)
Go back to XAMPP Control Panel and click Stop then Start next to Apache to restart it


Step 5 — Enable mod_rewrite in Apache
Laravel needs Apache's URL rewriting module to work.

In XAMPP Control Panel, click Config next to Apache → choose httpd.conf
Press Ctrl+F and search for:

   #LoadModule rewrite_module

Remove the # so the line reads:

   LoadModule rewrite_module modules/mod_rewrite.so

Now press Ctrl+F again and search for AllowOverride None. You will find a block that looks like this:

apache   <Directory "C:/xampp/htdocs">
       Options Indexes FollowSymLinks Includes ExecCGI
       AllowOverride None
       ...
Change AllowOverride None to AllowOverride All
Save the file and restart Apache in XAMPP Control Panel


Step 6 — Create the database

Open your browser and go to:

   http://localhost/phpmyadmin

Click New in the left sidebar
In the Database name field type exactly:

   lms_system

Set the collation to utf8mb4_unicode_ci
Click Create


Step 7 — Install Laravel dependencies

Open Command Prompt (search "cmd" in the Start menu)
Navigate into the project folder:

	cd C:\xampp\htdocs\learnify-laravel

Run:
	composer install

This downloads all of Laravel's packages into a vendor/ folder.


Step 8 — Create the environment file

Still in Command Prompt inside the project folder, run:
	copy .env.example .env


Step 9 — Generate the application key
In Command Prompt, still inside the project folder, run:
	php artisan key:generate
You should see:
	Application key set successfully.

Step 10 — Run migrations and seed the database
	php artisan migrate --seed
This will create all the tables and insert demo data. You should see a list of green DONE messages, one for each migration. When it finishes, open phpMyAdmin and refresh — you should see the tables inside lms_system.


Step 11 — Open the website
Go to your browser and visit:
	http://localhost/learnify-laravel/public
The Learnify homepage should appear.
